# InterviewAce AI Architecture

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Prisma
- PostgreSQL
- OpenAI
- React Hook Form
- Zod

## Project Structure

This project follows a feature-based architecture.

Business logic is separated into services.

Shared utilities are stored inside lib.

Reusable UI components are stored inside components.

Authentication, AI, Resume, Dashboard and Interview modules are isolated into independent domains.
